Селектори CSS

Як зв'язати HTML і CSS. Universal, Element, Class, ID — чотири типи що покривають 95% роботи. Чому клас — твій найкращий друг.

01 — Зв'язок HTML і CSS

Як CSS читає HTML

EdRow пояснює просто: HTML — стіни і дах, CSS — меблі і фарба. Селектор — це міст між ними. Ти вибираєш елемент у CSS і кажеш браузеру що з ним зробити.

Анатомія CSS правила
/*  СЕЛЕКТОР  {  ВЛАСТИВІСТЬ: ЗНАЧЕННЯ;  }  */

div {
  color: blue;          /* СИНІЙ текст */
}

/* Кілька правил */
.hero {
  color: red;
  padding: 100px;
}

/* Правило складається з:
   - Селектора → хто отримає стиль
   - Декларації → що зробити
   - Властивості → що змінити
   - Значення → на що змінити      */
02 — Чотири основних типи

Universal · Element · Class · ID

Ці чотири — 95% всієї роботи. EdRow каже: клас використовуй по-максимуму, ID — майже ніколи.

Universal Selector — *
/* Вибирає АБСОЛЮТНО ВСЕ */
* {
  padding: 0;
  margin: 0;
  box-sizing: border-box;
}

/* Використовується ОДИН РАЗ на початку
   для скидання браузерних дефолтів.
   Всі браузери мають свої стилі —
   ми їх скидаємо щоб контролювати все. */
Навіщо скидати margin/padding: браузери (Chrome, Firefox, Safari) додають свої стилі до елементів. h1 має margin зверху, ul — відступ зліва. Якщо не скинути — дивна поведінка на різних браузерах.
Element Selector — вибирає по тегу
/* Вибирає ВСІ елементи цього тегу */
div  { color: blue; }        /* всі div */
p    { font-size: 16px; }   /* всі p */
h1   { color: black; }      /* всі h1 */
body { background: #f7f7f7; } /* весь body */

/* Проблема: якщо є два header з різним стилем
   неможливо відрізнити — обидва отримають стиль */
Class Selector — найважливіший, точковий
/* Крапка перед іменем — це клас */
.hero         { padding: 100px; }
.hero__title  { color: black; }
.hero__desc   { color: #81da76; }

/* HTML */
<!-- В HTML: атрибут class="" -->
<header class="hero">
  <h1 class="hero__title">
    Empower Young Minds
  </h1>
  <p class="hero__desc">
    Innovative educational...
  </p>
</header>
Два класи на одному елементі: можна! <h2 class="feature__title feature__title--selected"> — перший клас дає базові стилі, другий додає модифікацію.
ID Selector — майже не використовується
/* Решітка перед іменем — це ID */
#hero-title { color: white; }

/* HTML: атрибут id="" */
<h1 id="hero-title">Empower Young Minds</h1>
Чому ID не використовують: ID може бути тільки ОДИН на сторінці. Ніколи не знаєш коли захочеш використати той самий стиль ще раз. EdRow: "Просто завжди використовуй клас — ніяких недоліків".

03 — Успадкування

Батько і дитина — ключова концепція

Кожен HTML елемент має батька і дітей. Якщо встановити колір на батькові — всі діти успадкують його. Це найважливіша концепція CSS.

Успадкування стилів
/* Встановлюємо шрифт на body */
body {
  font-family: 'Poppins', sans-serif;
  color: #333;
}

/* ВСІ елементи всередині body
   автоматично отримають Poppins!
   Не потрібно писати для кожного окремо */

.feature {
  color: blueviolet;
  /* .feature__title і .feature__desc
     теж стануть blueviolet */
}
Дерево батьків і дітей:
body ← встановлюємо font тут
section.feature
div.feature__container
h2.feature__title
← автоматично отримає шрифт
p.feature__desc
← і цей теж
04 — BEM

BEM — методологія іменування класів

EdRow згадує BEM коли пише hero__title. BEM вирішує проблему конфліктів імен у великих проєктах.

Block__Element--Modifier
/* BLOCK — незалежний компонент */
.hero              { }
.feature           { }
.video             { }

/* ELEMENT — частина блоку, __ (два підкреслення) */
.hero__title       { }   /* заголовок всередині hero */
.hero__description { }
.feature__item     { }
.video__container  { }

/* MODIFIER — варіант, -- (два дефіси) */
.feature__title--selected { }  /* виділений заголовок */
.cta__button--primary     { }  /* головна кнопка */
.cta__button--secondary   { }  /* другорядна кнопка */

/* Переваги: ніяких конфліктів!
   .hero__title ≠ .video__title
   Читаючи HTML — одразу розумієш структуру */
🎯 Практика
1. Відкрий VSCode, створи index.html і styles.css
2. В HTML: <header class="hero"> з h1 (class="hero__title") і p (class="hero__desc">
3. В CSS: .hero { padding: 100px; }, .hero__title { color: black; }, .hero__desc { color: #555; }
4. Встанови Live Server і перевір результат в браузері
