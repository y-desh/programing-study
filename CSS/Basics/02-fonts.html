Шрифти і Типографіка

font-family, font-size, font-weight, text-transform і всі текстові властивості. Різниця між px, em і rem.

01 — font-family

Як підключити і налаштувати шрифт

EdRow використовує Google Fonts (Poppins) — підключається через <link> в HTML. Встановлюємо шрифт на body — і всі дочірні елементи успадковують його автоматично.

Підключення Google Fonts
<!-- В <head> HTML файлу -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700&display=swap"
      rel="stylesheet">

/* CSS: застосовуємо на body */
body {
  /* Poppins — перший вибір, Times — fallback */
  font-family: 'Poppins', 'Times New Roman', serif;
}

/* Fallback шрифти — якщо Poppins не завантажився
   або недоступний — браузер використає Times */
Чому ставити на body, а не на кожен елемент
/* ❌ Погано — повторення */
.hero__title       { font-family: 'Poppins'; }
.hero__desc        { font-family: 'Poppins'; }
.feature__title    { font-family: 'Poppins'; }
.video__title      { font-family: 'Poppins'; }
/* і так далі... */
/* ✅ Добре — через успадкування */
body {
  font-family: 'Poppins', serif;
}
/* Всі елементи в body
   автоматично отримають Poppins.
   Один рядок замість десятків. */
02 — font-size: px vs rem vs em

Три одиниці для розміру шрифту

EdRow чітко пояснює різницю: px — абсолютний, не змінюється на різних екранах. rem — відносний до кореневого шрифту браузера, масштабується.

px, em, rem — порівняння
/* px — абсолютний, завжди 32px */
.feature__title { font-size: 32px; }

/* em — відносно розміру батька
   Проблема: вкладені em множаться! */
.feature__title { font-size: 2em; }
/* Якщо батько 16px → 2em = 32px
   Якщо батько 20px → 2em = 40px ← непередбачувано */

/* rem — відносно root (<html>), завжди стабільний */
.feature__title { font-size: 2rem; }
/* root (html) = 16px → 2rem = 32px завжди
   Незалежно від батьківського розміру! */

/* EdRow рекомендує rem для font-size:
   2.2rem ≈ 35px — хороший розмір для підзаголовка */
0.75rem = 12px
0.875rem = 14px
1rem = 16px (base)
1.25rem = 20px
1.5rem = 24px
2rem = 32px
3rem = 48px
03 — Текстові властивості

font-weight, text-transform та інші

Всі текстові властивості з App 1
.feature__title {
  font-size: 2.2rem;

  /* Товщина — число 100-900 або ключові слова */
  font-weight: 500;         /* 200=тонкий, 900=жирний */

  /* Трансформація — без зміни HTML */
  text-transform: uppercase; /* ВЕЛИКІ ЛІТЕРИ */

  color: #333;
  margin-bottom: 10px;
}

.feature__description {
  /* Вирівнювання тексту */
  text-align: center;        /* left | center | right */

  /* Міжлітерний інтервал */
  letter-spacing: 0.5rem;

  /* Висота рядка — без одиниці = множник */
  line-height: 1.6;           /* 1.6 × font-size */
  /* line-height: 2.4rem; — теж можна */

  /* Міжслівний інтервал */
  word-spacing: 3rem;

  /* Перенос рядків */
  white-space: nowrap;       /* без переносу */
}

.feature__title--selected {
  /* Підкреслення */
  text-decoration: underline;
}
Живе порівняння font-weight
font-weight: 200 — Light
font-weight: 300 — Thin
font-weight: 400 — Regular (default)
font-weight: 500 — Medium
font-weight: 600 — SemiBold
font-weight: 700 — Bold
font-weight: 800 — ExtraBold
font-weight: 900 — Black
Увага: щоб використовувати weight 300 або 700 — треба підключити їх в Google Fonts. wght@300;400;500;600;700 в URL. Якщо не підключив — браузер використає найближчий доступний.
line-height — міжрядковий інтервал
line-height: 1.2

Текст з вузьким міжрядковим інтервалом. Рядки близько один до одного. Підходить для заголовків але важко читати основний текст.

line-height: 1.8

Текст з комфортним міжрядковим інтервалом. Рядки мають простір між собою. Ідеально для основного тексту сторінки.

04 — Весь CSS для App 1 Fonts

Код з відео EdRow

features секція — повний CSS
/* ── FONTS SECTION ────────────────────────────── */

/* Шрифт для всього документа через body */
body {
  font-family: 'Poppins', 'Times New Roman', serif;
  background-color: #f7f7f7;
  color: #333;
}

.feature {
  padding: 100px;
}

.feature__item {
  padding: 30px 0;  /* 30px зверху і знизу, 0 по боках */
}

.feature__title {
  font-size: 2.2rem;
  font-weight: 500;
  text-transform: uppercase;
  color: #333;
  margin-bottom: 10px;
}

.feature__title--selected {
  /* Тільки перший заголовок з підкресленням */
  text-decoration: underline;
}

.feature__description {
  text-align: center;
  font-weight: 300;
  color: #6e6e6e;
}
🎯 Практика
1. Підключи Google Fonts (Poppins або будь-який на вибір) через <link> в head
2. Встанови font-family на body з fallback шрифтом
3. Зроби два h2: один 2rem bold, інший 1.5rem light italic
4. Параграф з line-height: 1.8 і color: #555
5. Кнопку з text-transform: uppercase і letter-spacing: 0.1em
