Кольори в CSS

HEX, RGB, RGBA, HSL — чотири способи задати колір. Правило 2-3 кольорів. Чому RGBA краще за просто RGB.

01 — Чотири формати

HEX · RGB · RGBA · HSL

CSS підтримує кілька форматів кольорів. EdRow найчастіше використовує HEX — його беруть з дизайн-інструментів. RGBA — коли потрібна прозорість.

HEX — шістнадцятковий формат
/* HEX: # + 6 символів (0-9, A-F) */
background-color: #55bb471;  /* зелений */
color: #333333;              /* темно-сірий */
color: #ffffff;              /* білий */

/* Скорочений запис: якщо пари однакові */
color: #fff;    /* = #ffffff */
color: #000;    /* = #000000 */
color: #f00;    /* = #ff0000 червоний */

/* EdRow: завжди використовуй пікер!
   VSCode підсвічує колір квадратом.
   Онлайн: coolors.co, adobe color */
#55bb47 — зелений CTA
#333 — основний текст
#f7f7f7 — фон (не чисто білий!)
#81da76 — акцентний текст
Чому не чисто білий #fff для фону? EdRow пояснює: чисто білий занадто "б'є в очі". Трохи сірий (#f7f7f7) виглядає приємніше для очей.
RGB і RGBA
/* RGB: Red Green Blue, кожен 0-255 */
color: rgb(32, 45, 50);

/* RGBA: + Alpha (прозорість) 0.0 - 1.0 */
background-color: rgba(0, 0, 0, 0.55);  /* чорний 55% */
background-color: rgba(0, 0, 0, 0.3);   /* чорний 30% */
background-color: rgba(0, 0, 0, 1.0);   /* непрозорий */

/* Головна перевага RGBA — можна контролювати
   прозорість кольору. Корисно для:
   - overlays на зображеннях
   - тіні
   - напівпрозорі фони
   - ефект затемнення */

/* EdRow використовує для фону hero-секції: */
.hero {
  background-color: rgba(0, 0, 0, 0.55); /* темний overlay */
  background-blend-mode: overlay;          /* змішування з фото */
}

HSL — Hue Saturation Lightness
/* HSL: hue(0-360) saturation(0-100%) lightness(0-100%) */
color: hsl(200, 50%, 50%);

/* Hue — відтінок (кут на колірному колі) */
/* 0° = червоний, 120° = зелений, 240° = синій */

/* Saturation — насиченість */
/* 0% = сірий, 100% = повна насиченість */

/* Lightness — яскравість */
/* 0% = чорний, 50% = нормальний, 100% = білий */

/* EdRow: в основному використовує HEX.
   HSL — коли треба варіанти одного кольору:
   змінив лише Lightness — темніший/світліший */
L:80%
L:60%
L:40%
L:25%

02 — Правило дизайну

2-3 кольори максимум

Правило від EdRow
❌ Помилка новачків
7+ кольорів = хаос
✅ Правильно
Primary · Dark · Light = гармонія
EdRow: «Використовуй якомога менше кольорів. Два-три основних — це вже достатньо. Більше кольорів → менш professional виглядає результат. Це те що роблять новачки — занадто багато кольорів.»
Код CTA секції з відео
/* ── COLORS SECTION ─────────────────────────── */
.cta {
  padding: 100px;
  background-color: #55bb471;   /* зелений фон */
  color: #fff;
  text-align: center;
}

.cta__title {
  font-size: 24px;
  font-weight: 400;
}

.cta__description {
  color: #d0d0d0;  /* трохи сірий на зеленому */
}

.cta__button {
  background-color: none;  /* скинути дефолт браузера */
  border: none;
  padding: 10px 20px;
  color: #fff;
  cursor: pointer;
}

.cta__button--primary {
  background-color: #333;  /* темна кнопка на зеленому */
}

.cta__button--primary:hover {
  background-color: #d0d0d0;
  cursor: pointer;
}
🎯 Практика
1. Обери свою палітру: 1 primary (яскравий), 1 dark (#333), 1 light (#f5f5f5)
2. Зроби секцію з зеленим/синім/фіолетовим фоном і білим текстом
3. Додай кнопку: прозорий фон, білий border, білий текст
4. На :hover — інвертуй: білий фон, темний текст
5. Перевір: чи три кольори поєднуються між собою?
