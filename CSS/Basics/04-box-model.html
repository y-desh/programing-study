Box Все є коробкою

Margin vs padding — найважливіша різниця. border, box-sizing: border-box, box-shadow. Кожен HTML елемент — прямокутна коробка.

01 — Концепція

Кожен елемент — коробка

EdRow: "Треба думати про будь-який HTML елемент як про коробку." Ця коробка складається з чотирьох шарів: content → padding → border → margin. Розуміння цього вирішує 90% проблем з layout.

Padding vs Margin — найважливіша різниця
/* PADDING — простір ВСЕРЕДИНІ елемента
   між вмістом і кордоном */
.video {
  padding: 100px;          /* з усіх боків */
  padding: 30px 0;        /* зверху/знизу ліво/право */
  padding: 10px 20px 30px 40px; /* top right bottom left */
}

/* MARGIN — простір ЗОВНІ елемента
   між ним і сусідніми елементами */
.video__description {
  margin-top: 10px;
  margin-bottom: 10px;
}

/* КЛЮЧОВА РІЗНИЦЯ:
   padding = частина коробки (всередині)
   margin = простір між коробками (зовні)

   EdRow: "padding — коли хочеш простір в середині,
   margin — коли хочеш простір між двома різними коробками" */
02 — Border і border-box

Border і box-sizing

border — рамка елемента
/* Скорочений запис: розмір стиль колір */
.video__button {
  border: 2px solid #55bb471;

  /* Стилі рамки: */
  border: 2px solid  #333; /* суцільна */
  border: 2px dashed #333; /* пунктирна */
  border: 2px dotted #333; /* точки */

  border-radius: 4px;   /* заокруглені кути */
  border-radius: 50%;   /* коло */
  border-radius: 100px; /* pill форма */
}

box-sizing: border-box — обов'язково!
/* ПРОБЛЕМА за замовчуванням (content-box): */
.box {
  width: 300px;
  padding: 20px;
  /* Реальна ширина = 300 + 20 + 20 = 340px! */
}

/* РІШЕННЯ — border-box: */
* { box-sizing: border-box; }
/* Тепер: */
.box {
  width: 300px;
  padding: 20px;
  /* Реальна ширина = РІВНО 300px
     padding включено в ширину ✅ */
}

/* EdRow завжди ставить це в * {} на початку:
   * { padding: 0; margin: 0; box-sizing: border-box; } */
03 — box-shadow

Тінь елемента

box-shadow — синтаксис і приклади
/* box-shadow: горизонталь вертикаль blur spread колір */
.video__container {
  box-shadow: 0 4px 8px 0 rgba(0,0,0,0.2);
}

/* horizontal — зміщення тіні вліво/вправо */
/* vertical   — зміщення вверх/вниз */
/* blur       — розмиття (більше = м'якша) */
/* spread     — розмір тіні (0 або мале = непомітне) */
/* color      — rgba для м'якшої тіні */
