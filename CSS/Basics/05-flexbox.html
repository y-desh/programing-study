Flexbox Одновимірний layout

display:flex, justify-content, align-items, flex-direction, flex-wrap. Коли NOT використовувати absolute — замість цього Flex.

01 — Коли Flex

Flexbox — одновимірний layout

EdRow: "Flexbox — це твій go-to для одновимірних layouts. Коли хочеш розмістити елементи в ряд або стовпець — використовуй Flex." Не absolute, не float — тільки Flex.

display: flex — вмикається на батьку
/* Flex вмикається на БАТЬКОВІ */
/* Впливає на ДІТЕЙ батька */

.programs {
  display: flex;

  /* Як розподілити дітей по головній осі */
  justify-content: space-between;  /* найпопулярніше */
  /* flex-start | flex-end | center | space-around | space-evenly */

  /* Вирівнювання по поперечній осі */
  align-items: center;    /* найпопулярніше! */
  /* flex-start | flex-end | stretch */

  gap: 20px;               /* відступ між дітьми */
}

/* Дитина */
.programs__card {
  width: 30%;             /* або flex-basis: 30% */
  height: 400px;
  background-color: #b5b5b5;
}
02 — justify-content

Розподіл по головній осі

Всі значення justify-content
flex-start (default)
center
space-between ← найчастіше!
space-around
03 — flex-direction і flex-wrap

Напрямок і перенос

flex-direction
.container {
  display: flex;

  /* Горизонтально (default) */
  flex-direction: row;

  /* Вертикально — для форм, sidebar */
  flex-direction: column;

  /* Перенос на новий рядок коли не вміщується */
  flex-wrap: wrap;    /* без цього — стискаються */
}
flex для дітей — flex-basis, flex-grow, flex (shorthand)
/* flex: grow shrink basis */
.element__left  { flex: 1; }  /* займає 1 частину */
.element__right { flex: 1; }  /* займає 1 частину */
/* Результат: обидва займають по 50% */

/* Або явно: */
.programs__card { flex-basis: 30%; }
/* = width: 30% але краще для flex контексту */

/* flex-grow: скільки вільного простору брати */
.card { flex-grow: 1; }
/* всі картки рівномірно заповнюють рядок */
Найпопулярніший Flex патерн від EdRow
/* NAVBAR: лого зліва, посилання справа */
.header__nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  max-width: 1500px;
  margin: 0 auto;
  padding: 0 4rem;
}

/* КАРТКИ в ряд з відступами */
.programs {
  display: flex;
  justify-content: space-between;
  gap: 20px;
}

/* ФОРМА вертикально */
.element__form {
  display: flex;
  flex-direction: column;
  gap: 20px;
}

/* EdRow: "justify-content: space-between
   і align-items: center — ці два я використовую
   найчастіше в Flex."                           */
Float застарів! EdRow чітко: "Float використовувався раніше, це був величезний головний біль. Сьогодні ніколи не використовуй float. Використовуй Flex замість нього завжди."
🎯 Практика
1. Зроби navbar: лого зліва, список посилань справа — через justify-content: space-between
2. Зроби секцію з 3 картками в ряд через display: flex і gap: 24px
3. Зроби картку вертикально: зображення, заголовок, текст, кнопка — flex-direction: column
4. Поекспериментуй з усіма значеннями justify-content і спостерігай різницю
