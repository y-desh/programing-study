Форми :hover · :focus · transition

input, select, label, button. Псевдокласи :hover і :focus. border-radius. Комбіновані селектори. Cursor pointer. Transition для smooth ефектів.

01 — Форми і HTML елементи

input, select, label, button

Форми — постійна частина будь-якого сайту. CSS для форм має особливості: треба скидати браузерні дефолти і додавати власні стилі.

HTML структура форми
<form class="element__form">
  <!-- label пов'язаний з input через for/id -->
  <div class="element__form-group">
    <label for="username">Ім'я</label>
    <input type="text"
           id="username"
           placeholder="Введіть ім'я">
  </div>

  <div class="element__form-group">
    <label for="course">Курс</label>
    <select id="course">
      <option>Виберіть курс</option>
      <option value="html">HTML & CSS</option>
      <option value="js">JavaScript</option>
    </select>
  </div>

  <button type="submit" class="element__button">
    Відправити
  </button>
</form>
02 — Псевдокласи і transition

:hover · :focus · transition

Стилі input і select
/* Комбінований селектор: form-group label */
.element__form-group label {
  font-weight: 500;
}

/* Кома — застосувати до ОБОХ */
.element__form-group input,
.element__form-group select {
  padding: 12px;
  border: 1px solid #e0e0e0;
  border-radius: 4px;
  font-family: inherit;  /* успадкувати від body! */
}

/* :focus — коли клікнули (отримало фокус) */
.element__form-group input:focus,
.element__form-group select:focus {
  outline: none;               /* прибираємо дефолтний */
  border-color: #34c759;        /* зелений border */
  box-shadow: 0 0 0 3px rgba(52,199,89,0.2);
}

.element__form-group select {
  cursor: pointer;   /* рука при наведенні */
  background-color: white;
}
Кнопка з hover і transition
.element__button {
  background-color: #34c759;
  color: white;
  padding: 12px 24px;
  border: none;         /* скинути дефолт */
  border-radius: 4px;
  cursor: pointer;
  font-weight: 500;

  /* transition — плавна зміна властивості */
  transition: background-color 0.2s;
  /* 0.2s = 200 мілісекунд — ледь помітно */
  /* 1s = 1 секунда — дуже повільно */
}

.element__button:hover {
  background-color: #2eb352;  /* трохи темніший */
  /* transition плавно анімує цю зміну */
}
font-family: inherit на input — важливо! Браузери за замовчуванням не успадковують шрифт для form елементів. Завжди додавай font-family: inherit до input, select і textarea.
🎯 Практика
1. Зроби форму: input для email, select з варіантами, кнопка submit
2. Стилізуй input: padding 12px, border 1px solid #ddd, border-radius 4px
3. Додай :focus стилі — інший border-color і легкий box-shadow
4. Кнопці додай transition і :hover ефект із зміною кольору
5. Перевір cursor: pointer на кнопці і select
