Зображення background-image · object-fit

background-image, background-size: cover, background-position, background-blend-mode. object-fit для img тегів. vh vs % для висоти.

01 — background-image

Фонове зображення для секції

EdRow використовує background-image для hero секції. Це не тег <img> — це CSS властивість яка додає зображення як фон.

hero з фоновим зображенням
.hero {
  height: 50vh;              /* 50% висоти вікна */
  /* vh = viewport height, незалежно від батька */
  /* % = від батька (body → залежить від контенту) */

  background-image: url('images/hero.jpeg');
  background-size: cover;    /* заповнити, обрізати */
  background-position: center; /* центрувати */

  /* Затемнення через rgba overlay */
  background-color: rgba(0, 0, 0, 0.55);
  background-blend-mode: overlay;
  /* blend-mode змішує колір і зображення */

  text-align: center;
  align-content: center;    /* вертикально центрує */
}

/* Після затемнення — текст треба зробити білим */
.hero__title {
  color: #f5f5f5;   /* майже білий на темному фоні */
}
background-size: cover vs contain
cover
Заповнює весь контейнер, може обрізати зображення. Для hero секцій.
contain
Вписується повністю, можуть бути порожні краї. Для логотипів.
02 — <img> і object-fit

Зображення в HTML через img тег

object-fit — як img заповнює контейнер
<!-- HTML -->
<div class="feature__image-container">
  <img class="feature__image"
       src="images/features.jpeg"
       alt="опис для незрячих">
</div>

/* CSS */
.feature__image {
  width: 100%;
  height: 100%;
  object-fit: cover;  /* не спотворювати! */
}

/* object-fit: cover — заповнює, обрізає якщо треба
   object-fit: contain — вписується повністю
   object-fit: fill — розтягує (спотворює!) */

/* Flex для розміщення поряд: */
.feature {
  display: flex;
  justify-content: space-between;
  gap: 20px;
}
.feature__container { flex-basis: 50%; }
.feature__image-container { flex-basis: 50%; }
03 — vh vs %

Viewport одиниці vs відсотки

Різниця vh і %
/* % — відносно БАТЬКА */
.hero {
  height: 50%;
  /* 50% від батька (body)
     body без height — контентна висота
     50% від нічого = нічого! */
}

/* vh — відносно VIEWPORT (вікна браузера) */
.hero {
  height: 50vh;
  /* Завжди 50% від висоти вікна
     Незалежно від батьків
     50vh = половина екрана */
}

/* 100vh = весь екран
   50vh = половина екрана
   vw = аналогічно, але ширина */

/* dvh (dynamic viewport height) — сучасніше:
   враховує мобільний browser bar */
.hero { height: 100dvh; }
🎯 Практика
1. Зроби hero секцію height: 60vh з background-image (або gradient як placeholder)
2. Додай background-color: rgba(0,0,0,0.5) і background-blend-mode: overlay
3. Текст героя зроби білим, text-align: center, align-content: center
4. Додай картку з img тегом — встанови width:100%, height:200px, object-fit:cover
