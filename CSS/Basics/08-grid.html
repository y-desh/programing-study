CSS Двовимірний layout

Коли Grid а не Flex — flowchart від EdRow. grid-template-columns, auto-fit, minmax, fr, span, grid-template-areas. Grid template areas для складних layouts.

01 — Flowchart: Flex чи Grid

Коли Grid, коли Flex

EdRow дає чіткий flowchart: одновимірний layout → Flex. Двовимірний layout → Grid. Grid значно спрощує складні макети і адаптивність.

Алгоритм вибору
? Це двовимірний layout (рядки І колонки)?
ТАК → Grid
Складний layout
→ grid-template-areas
Простий → grid-template-columns
НІ → одновимірний
Overlay/centric?
→ absolute/relative
Ні → Flex
02 — auto-fit і minmax

Адаптивна сітка без media queries

Магія auto-fit + minmax
/* Замість flex + wrap */
.latest-list {
  display: grid;

  /* auto-fit: скільки колонок влазить
     minmax(200px, 1fr): мін 200px, макс 1fr
     1fr = одна частина вільного простору */
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));

  gap: 2rem;
}

/* Результат:
   - широкий екран → 4-5 колонок
   - середній → 3 колонки
   - вузький → 2 → 1 колонка
   БЕЗ жодного @media query! */

/* auto-fit vs auto-fill:
   auto-fit  → розтягує наявні елементи
   auto-fill → залишає порожні місця */

03 — span і grid-template-areas

Складні layouts через Grid

span — елемент займає кілька колонок/рядків
/* 4 колонки, 2 рядки */
.hot-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(2, 1fr);
  gap: 1rem;
  height: 600px;
}

/* Елемент займає 2 колонки і 2 рядки */
.hot-grid__1 {
  grid-column: span 2;  /* = 2 колонки */
  grid-row: span 2;     /* = 2 рядки */
}

/* Або з явними позиціями: */
.hot-grid__1 {
  grid-column: 1 / 3;   /* від лінії 1 до 3 */
  grid-row: 1 / 3;       /* від лінії 1 до 3 */
}
grid-template-areas — найчитабельніший спосіб
/* EdRow: "Це найкращий спосіб для складних layouts
   — ти буквально бачиш структуру в коді" */

.gallery-grid {
  display: grid;
  grid-template-columns: repeat(5, 1fr);
  grid-template-rows: repeat(5, 150px);
  gap: 1rem;

  /* Карта layout — рядок = рядок сітки */
  grid-template-areas:
    "a a a a a"
    "a a a a a"
    "b c c d e"
    "b f f g e"
    "h h i i i";
}

/* Кожен елемент отримує area */
.gallery-item:nth-child(1) { grid-area: a; }
.gallery-item:nth-child(2) { grid-area: b; }
.gallery-item:nth-child(3) { grid-area: c; }
/* і так далі... */
🎯 Практика
1. Зроби картки через Grid: grid-template-columns: repeat(auto-fit, minmax(250px,1fr))
2. Зроби 4 блоки в 2x2 сітці — через grid-template-columns: repeat(2,1fr) і span
3. Спробуй grid-template-areas для простого layout: header, sidebar, main, footer
4. Порівняй складність Flex vs Grid для одного і того самого layout
