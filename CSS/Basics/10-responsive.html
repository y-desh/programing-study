Адаптивний @media queries

Media queries (@media screen and max-width). Mobile-first vs desktop-first. Grid і Flex вже роблять більшість адаптивності автоматично.

01 — Media Queries

@media — умовні стилі

Media queries дозволяють застосовувати різні стилі залежно від розміру екрана. EdRow: "Grid і Flex вже роблять багато автоматично. Media queries — для структурних змін."

Синтаксис @media
/* @media screen and (умова) { стилі } */
@media screen and (max-width: 768px) {
  /* Стилі для екранів < 768px */
  .header__links {
    display: none;  /* ховаємо навігацію */
  }
}

/* max-width — Desktop first (EdRow в цьому відео)
   min-width — Mobile first (краша практика) */

@media (max-width: 768px) {
  /* Зміна flex direction */
  .featured {
    flex-direction: column;
    height: auto;
    gap: 2rem;
  }

  /* Grid → block для мобільного */
  .masonry-grid {
    display: block;  /* всі в стовпець */
  }
  .masonry-item {
    margin-bottom: 1rem;
    height: 300px;
  }
}
02 — Mobile First

min-width vs max-width

Mobile First — рекомендований підхід
/* Desktop First (max-width) — EdRow в відео */
.grid {
  grid-template-columns: repeat(3, 1fr);
}
@media (max-width: 768px) {
  .grid { grid-template-columns: 1fr; }
}

/* ────────────────────────────────── */

/* Mobile First (min-width) — краща практика!
   Браузер завантажує мінімум CSS спочатку */
.grid {
  grid-template-columns: 1fr;  /* mobile */
}
@media (min-width: 640px) {
  .grid { grid-template-columns: repeat(2, 1fr); }
}
@media (min-width: 1024px) {
  .grid { grid-template-columns: repeat(3, 1fr); }
}

EdRow: Grid майже не потребує media queries
/* Магія auto-fit + minmax:
   АВТОМАТИЧНО адаптується без @media! */
.latest-list {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));
  gap: 2rem;
}

/* Широкий → 4 колонки
   Середній → 3 → 2 → 1 колонка
   Без жодного @media!  */

/* @media потрібен тільки для структурних змін:
   - ховати/показувати navbar
   - flex-direction: row → column
   - font-size для заголовків */
EdRow висновок: «Якщо ти правильно використовуєш Grid і Flex з самого початку — більшість адаптивності вирішується автоматично. @media для структурних змін, а не для кожного елемента.»
🎯 Фінальна практика
1. Відкрий DevTools (F12) → Ctrl+Shift+M → перевір адаптивність того що зробив
2. Додай @media (max-width: 768px) де Flex row → column для вузьких екранів
3. Сховай навігаційні посилання на мобільному через display: none
4. Перевір що grid з auto-fit адаптується без media queries
5. Задеплой на Hostinger або Netlify Drop — покажи світу результат!
