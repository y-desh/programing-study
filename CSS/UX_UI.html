01 — Відступи

Правила застосування відступів
Padding vs Margin
Padding — простір всередині елемента між вмістом і кордоном.
Margin — простір зовні між елементами.

Правило: padding для компонентів, gap для відстані між ними. Уникай margin між flex/grid дітьми — використовуй gap.
Вертикальний ритм
Пов'язані елементи → менший відступ (8–12px)
Групи елементів → середній (24–32px)
Розділи сторінки → великий (64–96px)

Чим ближчі елементи — тим більше вони пов'язані (Gestalt proximity).

Input → radius: 11px
Картка → radius: 6px
Всі різні → немає системи
✅ Послідовно
Кнопка → radius: 6px
Input → radius: 6px
Картка → radius: 12px

⚠️

03 — Лінії

Border Width

Товщина рамки — деталь яку часто недооцінюють. Надто товсті рамки "кричать". Надто тонкі — губляться. Правило: 1px для роздільників, 2px для акцентів, 0 для більшості.

05 — Міжрядковий

Висота рядка — один з найважливіших параметрів читабельності. Великий текст потребує менший line-height, малий — більший.

Правило: line-height без одиниціЗавжди пиши line-height: 1.6 без px чи rem — число автоматично множиться на поточний font-size. З px — може зламатися при зміні шрифту.
06 — Насиченість

08 — Тіні

Shadow Scale

Тінь створює відчуття глибини і ієрархії. Чим більша тінь — тим "вище" елемент над поверхнею. Правило: використовуй не більше 3 рівнів тіні на одній сторінці.
