01 — Змінні

CSS Custom Properties

CSS змінні — це не просто «константи». Вони живі: можуть змінюватися через JS, перевизначатися в дочірніх елементах, залежати від медіазапитів. Це фундамент будь-якої design system.

Синтаксис і область видимості
/* Оголошення — завжди починається з -- */
:root {                        /* :root = глобальна область */
  --color-primary: #a78bfa;
  --spacing-md: 16px;
  --radius: 8px;
  --shadow: 0 4px 20px rgba(0,0,0,0.2);
}

/* Використання */
.button {
  background: var(--color-primary);
  padding: var(--spacing-md);
  border-radius: var(--radius);
}

/* Fallback — якщо змінна не визначена */
.card {
  color: var(--color-text, #333); /* #333 = дефолт */
}

/* Локальне перевизначення */
.card-danger {
  --color-primary: #f87171;    /* тільки всередині .card-danger */
}
Темна/світла тема — лише через змінні
:root {
  --bg: #ffffff;
  --text: #1a1a2e;
  --surface: #f4f4f8;
}

[data-theme="dark"] {
  --bg: #0d0d1a;
  --text: #e2e4f0;
  --surface: #161722;
}

/* JS: */
document.documentElement
  .setAttribute('data-theme', 'dark');
Заголовок картки
Це текст всередині картки. Змінні CSS дозволяють перемикати теми без зміни жодного класу на самих компонентах.
Кнопка
Головна сила: змінна змінюється один раз — всі компоненти, що її використовують, оновлюються автоматично. Це і є design token.
02 — Псевдо

Pseudo-classes & Pseudo-elements

Pseudo-class (:) — стилізує елемент у певному стані. Pseudo-element (::) — стилізує або створює частину елемента.

Найважливіші pseudo-classes
:hover
При наведенні мишею
button:hover { opacity: 0.8 }
:focus
Коли елемент отримав фокус (tab / click)
input:focus { border-color: blue }
:active
Момент кліку (натиснена кнопка)
button:active { transform: scale(0.97) }
:nth-child(n)
Кожен N-й дочірній елемент
li:nth-child(odd) { background: #f5f5f5 }
:first-child / :last-child
Перший або останній серед сусідів
li:first-child { margin-top: 0 }
:not(selector)
Всі крім вказаних
li:not(:last-child) { border-bottom: 1px solid }
:is() / :where()
Групування без повтору. :where() не додає специфічності
:is(h1,h2,h3) { font-family: sans-serif }
:checked / :disabled
Стан форм-елементів
input:checked + label { color: green }
::before і ::after — найпотужніші pseudo-elements
/* Декоративна лінія під заголовком */
h2 {
  position: relative;
  padding-bottom: 12px;
}
h2::after {
  content: "";      /* обов'язково! */
  position: absolute;
  bottom: 0; left: 0;
  width: 48px;
  height: 3px;
  background: linear-gradient(
    to right, var(--v), var(--p)
  );
}

/* ::after  — ПІСЛЯ вмісту  */
/* content: "" — обов'язково */
/* position: absolute — часто */

03 — Рух

Transitions & Animations

transition — плавна зміна між двома станами (hover on/off). animation — незалежна послідовність ключових кадрів, що крутиться сама по собі.

transition — наведи, щоб побачити
opacity
:hover → 0
transform: scale
cubic-bezier bounce
transform: rotate
180deg
transform: translateX
+ color
background
+ border-radius
/* Формат transition */
.el {
  transition: властивість тривалість функція затримка;
  transition: transform 0.3s ease 0s;

  /* Кілька властивостей */
  transition: transform 0.3s ease,
              opacity 0.2s ease;

  /* НІКОЛИ не роби transition: all — це дорого */
  /* Вказуй конкретні властивості */
}
@keyframes animation — живе без hover
spin
linear infinite
bounce
cubic-bezier
pulse
box-shadow
wave
scaleX/Y
@keyframes spin {
  to { transform: rotate(360deg); }
}

.spinner {
  animation: spin 1s linear infinite;
  /*         name  dur timing  count  */
}

/* Затримка для stagger-ефекту */
.card:nth-child(1) { animation-delay: 0s;   }
.card:nth-child(2) { animation-delay: 0.1s; }
.card:nth-child(3) { animation-delay: 0.2s; }
Timing functions — швидкість руху
linear
ease (default)
ease-in
ease-out
cubic-bezier(bounce)
Правило: hover → ease-out (швидкий старт, плавна зупинка). Appear → ease. Механічні речі → linear.
GPU-friendly властивості — анімуй тільки transform і opacity. Вони не викликають reflow і перераховуються на GPU. Анімація width, margin, top/left — дорога операція, яка ламає 60fps.

05 — Колір

Кольори в CSS

CSS підтримує десятки форматів кольору. Знати різницю між hex, rgb, hsl і oklch — не академізм, а практична необхідність.

Формати кольорів
#a78bfa
HEX — найпоширеніший. Не можна змінити прозорість.
rgba(167, 139, 250, 0.5)
RGB + Alpha — є прозорість. Незручно редагувати.
hsl(258, 89%, 76%)
HSL — зрозуміло для людини. Легко робити варіації.
hsl(258 89% 76% / 0.4)
Сучасний синтаксис — без ком, slash для alpha.
oklch(72% 0.15 295)
oklch — майбутнє CSS. Перцептуально рівномірний.
HSL — найзручніший для дизайнера
/* HSL = Hue Saturation Lightness */
/*        відтінок насич. яскрав. */

/* Один відтінок — різна яскравість */
--blue-100: hsl(220 90% 95%);
--blue-500: hsl(220 90% 55%);
--blue-900: hsl(220 90% 15%);

/* Зміна лише яскравості = палітра! */

oklch — новий стандарт. На відміну від HSL, де два кольори з однаковою "яскравістю" виглядають по-різному яскраво, oklch перцептуально рівномірний. Підтримується у всіх сучасних браузерах з 2023.
06 — Адаптивність

Media Queries

Media queries дозволяють застосовувати різні стилі залежно від характеристик пристрою: ширини екрана, орієнтації, теми системи. Основа responsive design.

Стандартні breakpoints (Tailwind-стиль)
НазваMedia QueryCSS змінна / застосування
sm@media (min-width: 640px)Великий телефон, починаємо розширювати
md@media (min-width: 768px)Планшет, 2 колонки
lg@media (min-width: 1024px)Ноутбук, sidebar з'являється
xl@media (min-width: 1280px)Десктоп, повна ширина контенту
2xl@media (min-width: 1536px)Широкий монітор
Mobile-first vs Desktop-first
✅ Mobile-first (правильно)
/* За замовч — мобільний */
.grid {
  grid-template-columns: 1fr;
}

/* Розширюємо для більших */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: 1fr 1fr;
  }
}
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
❌ Desktop-first (застаріло)
/* За замовч — десктоп */
.grid {
  grid-template-columns: repeat(3, 1fr);
}

/* Зменшуємо для малих */
@media (max-width: 1024px) {
  .grid {
    grid-template-columns: 1fr 1fr;
  }
}
@media (max-width: 768px) {
  .grid { grid-template-columns: 1fr; }
}
Нестандартні media features
/* Темна тема операційної системи */
@media (prefers-color-scheme: dark) {
  :root { --bg: #0d0d1a; --text: #e2e4f0; }
}

/* Вимкнути анімації (accessibility) */
@media (prefers-reduced-motion: reduce) {
  * { animation-duration: 0.01ms !important; }
}

/* Орієнтація пристрою */
@media (orientation: landscape) {
  .hero { min-height: 50vh; }
}

/* Тільки екран (не принтер) */
@media screen { ... }
@media print  { .sidebar { display: none; } }
Сучасна альтернатива: замість media queries для сіток часто достатньо grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)) — адаптується автоматично без єдиного breakpoint.
07 — Функції

CSS Functions

CSS — не просто декларації. Вбудовані функції дозволяють обраховувати значення, адаптувати розміри і створювати складні ефекти без жодного рядка JavaScript.

calc()
calc(вираз)
Математика зі змішаними одиницями. Одна з найстаріших і найважливіших.
width: calc(100% - 64px); padding: calc(var(--spacing) * 2); height: calc(100vh - 80px);
clamp()
clamp(min, ideal, max)
Значення плавно масштабується між мінімумом і максимумом. Замінює media queries для типографіки.
font-size: clamp(1rem, 4vw, 2rem); padding: clamp(16px, 5vw, 48px);
min() / max()
min(a, b) / max(a, b)
Повертає менше або більше з двох значень. Елегантна альтернатива media queries для ширини.
width: min(100%, 600px); /* = max-width: 600px, але коротше */ font-size: max(14px, 1.5vw);
var()
var(--name, fallback)
Читає CSS custom property. Другий аргумент — fallback якщо змінна не визначена.
color: var(--color-text, #333); margin: var(--spacing-md, 16px);
rgb() / hsl()
hsl(hue sat% light% / alpha)
Кольорові функції з опціональним alpha каналом в сучасному синтаксисі.
color: hsl(220 90% 60% / 0.8); background: rgb(167 139 250 / 0.1);
linear-gradient()
linear-gradient(dir, ...colors)
Градієнт як значення background. Можна накладати кілька через кому.
background: linear-gradient(rgba(0,0,0,0.4), transparent), url('hero.jpg') center/cover;
Практичні комбінації
/* Адаптивний padding без media queries */
.section {
  padding: clamp(40px, 8vw, 120px);
}

/* Контейнер з відступами */
.container {
  width: min(100% - 48px, 1200px);
  margin-inline: auto;
}

/* Динамічна висота з вирахуванням navbar */
.page {
  min-height: calc(100vh - var(--navbar-height));
}

/* Fluid typography — адаптивна шкала */
:root {
  --text-sm:  clamp(0.8rem,  1.5vw,  0.9rem);
  --text-md:  clamp(1rem,    2vw,    1.1rem);
  --text-lg:  clamp(1.2rem,  3vw,    1.5rem);
  --text-xl:  clamp(1.5rem,  4vw,    2.5rem);
  --text-2xl: clamp(2rem,    6vw,    4rem);
}
