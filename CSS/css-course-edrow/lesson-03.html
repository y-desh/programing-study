Colors
Кольори в CSS

hex, rgb, rgba, hsl — всі чотири формати. Чому #55bb47, rgba для прозорості і золоте правило: не більше 2–3 кольорів на сайті.

01 — Формати кольорів

Hex, RGB, RGBA, HSL

Ed Row будує CTA (Call to Action) секцію і пояснює всі способи задати колір в CSS. Є чотири основних формати. У більшості часу ти будеш використовувати hex.

1 · Hex — найпоширеніший
.cta {
  /* #55bb47 — зелений колір Ed Row */
  background-color: #55bb47;

  /* Структура: #rrggbb             
     rr = red   (00–ff)              
     gg = green (00–ff)              
     bb = blue  (00–ff)              */

  /* Скорочення (якщо пари однакові): */
  color: #fff;    /* = #ffffff — білий */
  color: #000;    /* = #000000 — чорний */
  color: #333;    /* = #333333 — темно-сірий */

  /* В VS Code: наведи на колір —     
     з'явиться Color Picker         */
}
#55bb47 — green (Ed Row)
#f7f7f7 — off-white (body bg)
#333 = #333333 — dark text
Color Picker в VS Code Напиши hex код → наведи курсор → з'явиться квадратик кольору. Натисни — відкриється вибір кольору з можливістю перетягувати picker.
2 · RGB — три числа 0–255
/* RGB: три значення 0–255 для red, green, blue */
.example {
  background-color: rgb(85, 187, 71);
  /*                    ↑     ↑     ↑
                      red  green  blue
                      
   rgb(0, 0, 0)   = чорний
   rgb(255,255,255) = білий
   rgb(255, 0, 0)  = червоний
   rgb(0, 255, 0)  = зелений
   rgb(0, 0, 255)  = синій      */
}
3 · RGBA — з прозорістю ★ (найкорисніший)
/* A = Alpha (прозорість): 0 = прозорий, 1 = непрозорий */
.overlay {
  background-color: rgba(0, 0, 0, 0.55);
  /* чорний 55% прозорості             */
  /* Ed Row використовує це для hero   */
  /* фонового зображення               */
}

/* Приклади прозорості: */
rgba(0, 0, 0, 0.0)   /* повністю прозорий */
rgba(0, 0, 0, 0.3)   /* 30% видимий     */
rgba(0, 0, 0, 0.7)   /* 70% видимий     */
rgba(0, 0, 0, 1.0)   /* повністю видимий */

/* Чому rgba краще ніж opacity?       
   opacity: 0.5 робить прозорим       
   ВЕСЬ елемент включно з вмістом.   
   rgba — тільки колір фону.         */

4 · HSL — Hue Saturation Lightness
/* HSL: відтінок (0-360°), насиченість (%), яскравість (%) */
.example {
  color: hsl(200, 50%, 50%);
  /*         ↑    ↑      ↑
            hue  sat   light
            
   hue: 0=red, 120=green, 240=blue, 360=red
   saturation: 0% = сірий, 100% = яскравий
   lightness: 0% = чорний, 50% = нормальний, 100% = білий  */
}

/* Зручно для варіацій одного кольору: */
hsl(120, 47%, 30%)  /* темно-зелений */
hsl(120, 47%, 51%)  /* нормальний зелений */
hsl(120, 47%, 70%)  /* світло-зелений */
/* Один hue, різна яскравість — легко! */
02 — Кольори проєкту

CSS для CTA секції та кнопки

Ed Row каже важливе правило: "Use as few colors as possible — probably two or three main colors for any website." Більше — виглядає аматорськи.

CTA секція — повний CSS
styles.css — CTA section
/* ── SECTION: Colors ───────────────────────────── */

.cta {
  background-color: #55bb47; /* зелений фон */
  color: #fff;                  /* білий текст */
  padding: 100px;
  text-align: center;
}

.cta__description {
  color: #e0e0e0; /* трохи сірий на зеленому */
}

/* Кнопка — скидаємо браузерні стилі */
.cta__button {
  background-color: none;
  border: 2px solid white;
  color: white;
  padding: 10px 20px;
  cursor: pointer;
}

.cta__button--primary {
  background-color: white;
  color: #55bb47; /* інвертовані кольори */
}

«For design elements, the most important thing is use as few colors as possible — probably like two or three main colors that you use for any kind of website. That is what a lot of amateurs end up doing — using too many colors.»

— ED ROW
Іменовані кольори — є, але не зловживай

color: red;
color: white;
color: black;

color: blanched-almond;    /* теж є! */
color: cornflower-blue;
color: medium-aquamarine;

/* Але в реальних проєктах — завжди hex.  
   Іменовані — для швидкого тестування. */
🎯 Завдання — Лекція 03
1. Створи CTA секцію з зеленим фоном (#55bb47) і білим текстом
2. Додай кнопку — скинь браузерні стилі (border: none, background: none), потім постилізуй
3. Поекспериментуй з rgba(0,0,0, різні значення) — накладай на будь-який елемент
4. Спробуй hsl — зміни тільки третій параметр (lightness) щоб отримати темніші/світліші варіації
5. Відкрий Color Picker VS Code — клікни на квадратик кольору і поекспериментуй
