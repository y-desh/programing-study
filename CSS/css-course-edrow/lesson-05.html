Flexbox
Одновимірний layout

display: flex, justify-content (space-between, center), align-items, flex-direction, gap, flex-basis, flex: 1. Навбар і три колонки.

01 — Що таке Flex

display: flex — одновимірний layout

Ed Row: "Flexbox is a nice way to lay out a lot of elements. The most common flex example: display flex on the parent and then you can set justify-content space-between."

Базова концепція Flex
/* display: flex ставиться на БАТЬКА */
<div class="programs">          ← батько
  <div class="programs__card"></div>
  <div class="programs__card"></div>
  <div class="programs__card"></div>
</div>

.programs {
  display: flex;              ← батько
  justify-content: space-between;
  gap: 20px;
}

.programs__card {           ← дитина
  flex-basis: 30%;
  height: 400px;
  background: #b5b5b5;
  padding: 30px;
}

02 — justify-content

Розміщення по основній осі

Всі значення justify-content
.programs {
  display: flex;
  justify-content: space-between; ← найчастіший
}

/* Значення justify-content: */
flex-start    /* всі зліва (default) */
flex-end      /* всі справа          */
center        /* всі по центру       */
space-between /* перший зліва, останній справа, решта рівномірно */
space-around  /* рівні відступи навколо кожного */
space-evenly  /* рівні відступи між всіма       */
space-between:
center:
flex-start:

03 — align-items

Вирівнювання по поперечній осі

align-items — вертикальне центрування
.programs {
  display: flex;
  justify-content: space-between;
  /* align-items Centre — вирівнює по вертикалі */
  align-items: center;
  /* Якщо height батька: 700px —
     елементи будуть по центру HEIGHT */
}

/* Значення align-items: */
flex-start  /* верх */
center      /* центр ← найчастіший */
flex-end    /* низ  */
stretch     /* розтягнути до висоти батька (default) */
baseline    /* по базовій лінії тексту */

Найпоширеніший патерн у Flexbox justify-content: space-between + align-items: center — Ed Row використовує це скрізь. Навбар, три колонки, будь-який горизонтальний layout.
04 — flex-direction і flex: 1

Напрям і розміщення

flex-direction — горизонталь чи вертикаль
/* flex-direction: row (default) — ліво направо */
.programs { flex-direction: row; }

/* flex-direction: column — зверху вниз */
.element__form {
  display: flex;
  flex-direction: column;
  /* Тепер gap між input полями вертикальний */
  gap: 20px;
}

/* flex: 1 — займає всю доступну ширину */
.element__left  { flex: 1; }
.element__right { flex: 1; }
/* Обидва займуть по 50% (одна одиниця кожен) */

/* flex-basis: 30% — ширина елемента у flex */
.programs__card { flex-basis: 30%; }
/* Аналогічно width: 30% але "правильніше" в flex */

/* float — ЗАСТАРІЛО! Ed Row: "Never use float." */
/* float: left — не використовуй, це 2010 рік   */
Float — застаріло Ed Row прямо каже: "Float used to be the old way of positioning things. It was a huge pain. Nowadays — never use float. It is something I would consider deprecated."
Форма секції — flex column з gap
/* Дві колонки поряд */
.element {
  display: flex;
  justify-content: space-between;
  gap: 60px;
  padding: 100px;
}
.element__left  { flex: 1; } ← 50%
.element__right { flex: 1; } ← 50%

/* Форма — вертикальний flex */
.element__form {
  display: flex;
  flex-direction: column;
  gap: 20px;
}

/* Форм-група (label + input) */
.element__form-group {
  display: flex;
  flex-direction: column;
  gap: 8px;
}
🎯 Завдання — Лекція 05
1. Для .programs: display flex, justify-content space-between, gap 20px
2. Для .programs__card: flex-basis 30%, height 400px, background #b5b5b5
3. Для .element: display flex, gap 60px. Для .element__left і .element__right: flex 1
4. Поекспериментуй з justify-content — space-between, center, flex-end
5. Поекспериментуй з align-items — помітиш різницю тільки якщо у батька задана height
