01 — HTML форми

input, select, label, button

Форми — особливий випадок в CSS. Браузери додають власні дефолтні стилі до input, select, button які треба спочатку скинути, а потім написати свої.

HTML структура форми
index.html
<form class="element__form">
  <!-- label + input пов'язані for/id -->
  <div class="element__form-group">
    <label for="username">Username</label>
    <input
      type="text"
      id="username"
      placeholder="Enter username"
    >
  </div>

  <div class="element__form-group">
    <label for="course">Course</label>
    <select id="course">
      <option value="">Choose a course</option>
      <option value="html">HTML & CSS</option>
      <option value="js">JavaScript</option>
      <option value="react">React</option>
    </select>
  </div>

  <button type="submit"
          class="element__button">
    Submit Form
  </button>
</form>
02 — CSS для форм

Стилізація inputs і :focus псевдоклас

Повний CSS для форми секції
styles.css — element / form section
/* ── SECTION: Form Elements ─────────────────────── */

/* Дві колонки */
.element {
  display: flex;
  justify-content: space-between;
  gap: 60px;
  padding: 100px;
  background-color: #f8f9fa;
}
.element__left  { flex: 1; }
.element__right { flex: 1; }

/* Форма — вертикальна */
.element__form {
  display: flex;
  flex-direction: column;
  gap: 20px;
}

/* Група label + input */
.element__form-group {
  display: flex;
  flex-direction: column;
  gap: 8px;
}

/* Мітка */
.element__form-group label {
  /* Комбінований селектор!             
     .element__form-group + label      
     = label тільки всередині .element__form-group */
  font-weight: 500;
}

/* Input і Select — однакові стилі через кому */
.element__form-group input,
.element__form-group select {
  padding: 12px;
  border: 1px solid #e0e0e0;
  border-radius: 4px;
  font-family: inherit; ← КРИТИЧНО!
}

/* :focus — стан коли клікнув на поле */
.element__form-group input:focus,
.element__form-group select:focus {
  outline: none;           /* прибираємо стандартний */
  border-color: #34c759;  /* зелений  */
  box-shadow: 0 2px rgba(52,199,89,0.2);
}

/* Select специфічне */
.element__form-group select {
  cursor: pointer;
  background-color: white;
}

/* Кнопка submit */
.element__button {
  background-color: #34c759;
  color: white;
  padding: 12px 24px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  font-weight: 500;
  transition: background-color 0.2s;
}

/* Hover ефект */
.element__button:hover {
  background-color: #2db350; ← трохи темніший
}
Три ключові концепти лекції 06
1. font-family: inherit
Input і select не успадковують font-family автоматично — в них є свій дефолтний системний шрифт. Завжди додавай font-family: inherit щоб вони виглядали однаково з рештою сторінки.
2. Комбінований селектор
.element__form-group input { /* пробіл між ними */
  /* input ВСЕРЕДИНІ .element__form-group */
  /* не ВЕСЬ input на сторінці           */
}

.element__form-group input,
.element__form-group select {
  /* обидва через кому — однакові стилі */
}
3. transition — плавні переходи
.element__button {
  transition: background-color 0.2s;
  /* при hover — background зміниться плавно */
  /* 0.2s = 200 мілісекунд                  */
}
.element__button:hover {
  background-color: #2db350;
  /* зміна відбувається плавно за 0.2с     */
}
🎯 Завдання — Лекція 06
1. Зроби .element__form з flex-direction column і gap 20px
2. Зроби .element__form-group з flex-direction column і gap 8px
3. Через комбінований селектор стилізуй input і select: padding 12px, border, border-radius 4px, font-family inherit
4. Додай :focus стиль — outline none, зелений border-color, box-shadow
5. Кнопці додай transition background-color 0.2s і :hover з трохи темнішим кольором
6. Перевір: cursor: pointer на select і button (мусить бути рука при наведенні)
Підказка: cursor: pointer важливий — без нього select виглядає як звичайний текст
