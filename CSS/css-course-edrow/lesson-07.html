01 — background-image

Hero секція з фоновим зображенням

Ed Row робить hero красивим — додає фотографію на фон і темний оверлей щоб текст залишався читабельним.

CSS для hero з фото
styles.css — hero background
/* Додаємо до вже існуючого .hero */
.hero {
  height: 50vh;  /* 50% висоти вікна браузера */

  background-image: url('images/hero.jpeg');
  background-size: cover;     /* заповнює всю площу */
  background-position: center; /* центрує зображення */

  /* Темний оверлей через rgba */
  background-color: rgba(0, 0, 0, 0.55);
  background-blend-mode: overlay;

  text-align: center;
  align-content: center;
}

/* Тепер текст білий */
.hero__title {
  color: #f5f5f5; ← замість black
}
Empower Young Minds
Innovative educational platform
Симуляція: темний оверлей через background-blend-mode: overlay
background-size і background-position
background-size: cover;
/* Заповнює контейнер — може обрізати з країв.
   Зображення завжди покриває всю площу. ← Ed Row */

background-size: contain;
/* Вписується в контейнер — можуть бути пробіли. */

background-position: center;  /* по центру */
background-position: top;     /* верхня частина */
background-position: 50% 30%; /* точне позиціонування */
Чому vh, а не %? height: 50% = 50% від БАТЬКА (а батько не має висоти). height: 50vh = 50% від висоти ВІКНА браузера. vh — завжди від viewport, незалежно від батька.
02 — object-fit

Зображення в контейнері фіксованого розміру

Зображення у картках програм
.programs__card {
  background: none;          ← прибираємо сірий фон
  padding: 0;
  flex-basis: 30%;
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  height: 500px;
  border-radius: 8px;
}

.programs__card .image {
  width: 100%;
  height: 70%;
  min-height: 200px;  ← мінімальна висота
  object-fit: cover;  ← заповнює без спотворення
  border-radius: 4px;
}

/* margin-top: auto — кнопка прилипає до низу */
.programs__button {
  margin-top: auto;
  background-color: #34c759;
  color: white;
  padding: 10px 20px;
  border-radius: 4px;
  cursor: pointer;
}
object-fit: cover
заповнює весь контейнер ← використовуй
object-fit: contain — пробіли
