Flex vs Grid
Коли що використовувати?

Флоу-чарт рішення: 2D → Grid, overlay → position: absolute/relative, 1D → Flex. App 2 навбар і featured секція через Flex.

01 — Алгоритм рішення

Флоу-чарт: Flex, Grid або Position?

Ed Row: "This is one of the most important parts of understanding CSS — a lot of people overlook this aspect." Є чіткий алгоритм. Вчи його напам'ять.

Три запитання — три рішення
🔲 КРОК 1: Чи потрібен 2D layout?
2D = елементи займають і рядки, і колонки одночасно. Приклади: новинна сітка де блок займає 2 рядки і 2 колонки, photo gallery, масонрі. Якщо бачиш "мозаїку" — це 2D.
ТАК → Grid
НІ → Крок 2
📌 КРОК 2: Це overlay або centering?
Overlay = один елемент поверх іншого. Текст на зображенні, badge у кутку картки, modal, tooltip. Якщо щось "плаває" поверх — це position.
ТАК → position: relative + absolute
НІ → Крок 3
➡️ КРОК 3: Одновимірне = Flexbox
Елементи в рядок або стовпець. Навбар, форма, список карток, будь-який лінійний layout. Flex — перший вибір для 1D.
→ Flex

«A lot of people are always tempted to use absolute and relative positioning for their layout concerns — but it should be their last option. Float is deprecated — never use it. Flexbox is your go-to first option for one-dimensional layouts.»

— ED ROW

.header {
  background: #1d1d1d;
  color: #fff;
  padding: 1.5rem 0;
}

.header__nav {
  display: flex;
  justify-content: space-between;

  padding: 0 4rem;

.header__logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.header__links {
  display: flex;
  gap: 2rem;
}

.header__links a {
  color: #fff;
  text-decoration: none;
  cursor: pointer;
}
.header__links a:hover {
  color: #9c9c9c;

margin: 0 auto — центрування контейнера Ed Row пояснює: "margin: 0 auto basically centers our left and right margin." Використовується разом з max-width щоб контент не розтягувався на весь екран.
03 — Featured article

Flex горизонтальний + вкладений Flex column

Зображення + текст поряд, контент вертикально
/* Зображення і текст поряд = Flex */
.featured {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 2rem;
  height: 600px;
}

.featured__image {
  flex-basis: 50%;
  height: 100%;
  object-fit: cover;
}

/* Текстовий блок — ще один flex, але вертикальний */
.featured__content {
  flex: 1;
  display: flex;
  flex-direction: column;
  justify-content: center;
  gap: 1rem;
}

/* Кнопка з hover */
.featured__more {
  background: #1d1d1d;
  color: #fff;
  padding: 1rem 2rem;
  border-radius: 5px;
  border: none;
  cursor: pointer;
}
.featured__more:hover {
  background: #414141;
  color: #d9d8d8;
  cursor: pointer;
}

/* footer between items */
.featured__footer {
  display: flex;
  align-items: center;
  gap: 1rem;
  margin-top: 2rem;
}
Вкладені Flex — нормально! Ed Row: "You can have a lot of different layers for your layout — nested flexes." .featured — горизонтальний flex. .featured__content всередині — вертикальний flex. Це поширена практика.
