CSS Grid
Двовимірний layout

display: grid, grid-template-columns, repeat, auto-fit, minmax, fr, grid-column span, grid-template-areas. Адаптивна сітка без media queries.

01 — Основи Grid

grid-template-columns — визначаємо колонки

Ed Row: "Grid is a much more useful way to do two-dimensional layouts, especially for complex layouts."

Базовий Grid синтаксис
.latest__list {
  display: grid;

  /* 3 рівні колонки через repeat */
  grid-template-columns: repeat(3, 1fr);
  /*                       ↑     ↑   ↑
                    repeat 3  рівні
                    разів   колонки  */

  gap: 2rem;
}

/* fr = fractional unit (частка вільного простору) */
/* 1fr 1fr 1fr = кожна по 33.3%                   */
/* 2fr 1fr     = 66% і 33%                         */
/* 280px 1fr   = 280px і решта                     */

/* Hot Grid: 4 колонки × 2 рядки */
.hot-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(2, 1fr);
  gap: 1rem;
  height: 600px;
}

02 — auto-fit і minmax

Адаптивна сітка БЕЗ media queries

Ed Row: "This property is exactly why I recommend Grid — it makes responsive design a lot easier." Один рядок замість 5 media queries.

repeat(auto-fit, minmax()) — магічний рядок
.latest__list {
  display: grid;

  /* auto-fit: браузер сам визначає кількість колонок */
  /* minmax(200px, 1fr): мінімум 200px, максимум 1fr  */
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));

  gap: 2rem;
}

/* Як це працює:
   При ширині 1200px → 6 колонок по 200px
   При ширині 800px  → 4 колонки
   При ширині 400px  → 2 колонки
   Responsive без жодного @media query! ✅ */

/* auto-fit  — колонки розтягуються щоб заповнити */
/* auto-fill — залишає порожні місця               */
/* Ed Row рекомендує auto-fit                      */

03 — grid-column span

Елемент займає кілька комірок

Hot Grid — 4 колонки, 2 рядки
.hot-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(2, 1fr);
  gap: 1rem;
  height: 600px;
}

/* Перший — займає 2 колонки і 2 рядки */
.hot-grid-1 {
  grid-column: span 2;  /* або: 1 / 3 */
  grid-row: span 2;
}

/* Другий і третій — стандартна 1×1 комірка */
.hot-grid-2, .hot-grid-3 {
  grid-column: span 1;
  grid-row: span 1;
}

/* Четвертий — 2 колонки, 1 рядок */
.hot-grid-4 {
  grid-column: span 2;
}

/* span 2 = займає 2 юніти від поточної позиції  */
/* 1 / 3  = від лінії 1 до лінії 3 (= 2 колонки) */

grid-template-areas — іменовані зони
/* Для дуже складних layouts — іменовані зони */
.gallery-grid {
  display: grid;
  grid-template-columns: repeat(5, 1fr);
  grid-template-rows: repeat(5, 150px);
  /* Для висоти — абсолютні px! Ed Row:   
     "For height use absolute units."   */

  /* Візуальна карта layout прямо в CSS! */
  grid-template-areas:
    "a a b c d"
    "a a e e d"
    "f g e e h"
    "f g i j h"
    "k k l m n";
}

/* Кожен елемент отримує ім'я зони */
.gallery-item-a { grid-area: a; }
.gallery-item-b { grid-area: b; }
/* ... і так далі */

.gallery-item img {
  width: 100%; height: 100%;
  object-fit: cover;
}

Чому template-areas — кращий підхід для складних layouts Ed Row: "This makes it easy to come back to. You want to always take into consideration that there is an issue later on to maintain — and this is a huge issue especially when it comes to AI code."
