Workflow — порядок роботи

1
Figma темплейт
Pages: _Tokens, _Cover, Landing. Фрейми: lg (1440px), md (768px), sm (390px). Layout grid на кожному.
2
Токени і компоненти
Кольори як Color Styles. Типографія як Text Styles. Кнопки як компоненти з Variants.
3
Vite проєкт
npm create vite@latest → встанови Tailwind v4 → налаштуй vite.config.js → очисти файли.
4
style.css — токени
@theme для кольорів і шрифту. @layer components для кнопок. @layer base для body стилів.
5
Верстка секція за секцією
Navbar → Hero → Features → Social Proof → CTA → Footer. Коментарі між секціями.
6
Адаптивність
Mobile-first. Відкрий DevTools → 390px. Додавай lg: префікси для десктопу.
7
JS функціональність
Бургер меню toggle. Smooth scroll. Sticky navbar тінь при скролі.
8
Деплой
GitHub Pages або Netlify. npm run build → папка dist → деплой.
