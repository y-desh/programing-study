01

accent-color — колір браузерних елементів

utility

Змінює колір дефолтних браузерних елементів — чекбокси, radio, range слайдери. Без додаткового CSS або бібліотек.

HTML 
<!-- Звичайний синій браузера -->
<input type="checkbox" checked>

<!-- Кастомний колір -->
<input type="checkbox" class="accent-sky-400" checked>
<input type="checkbox" class="accent-emerald-400" checked>
<input type="checkbox" class="accent-violet-500" checked>
<input type="range"    class="accent-orange-400">

02

Fluid text — clamp() без breakpoints

typography

Замість стрибків на breakpoints — текст змінюється плавно на кожен піксель екрану. Resize вікно браузера щоб побачити різницю.

HTML 
<!-- Звичайний — стрибає на breakpoints -->
<h1 class="text-2xl md:text-4xl lg:text-6xl">
  Заголовок
</h1>

<!-- Fluid — плавно змінюється -->
<h1 class="text-[clamp(24px,5vw,72px)]">
  Заголовок
</h1>

<!-- Формула: clamp(мін, поточний, макс) -->
<p class="text-[clamp(14px,2vw,20px)]">
  Параграф що плавно масштабується
</p>

03

file: — стилізація input type="file"

utility

Префікс file: застосовує стилі до кнопки вибору файлу. Без кастомних бібліотек і хаку з display:none.

HTML 
<input
  type="file"
  class="
    file:mr-4 file:py-2 file:px-6
    file:border-0 file:rounded-lg
    file:bg-sky-500 file:text-white
    file:font-medium file:cursor-pointer
    file:hover:bg-sky-600
    file:transition-colors
    text-slate-400 text-sm
    cursor-pointer
  "
>

04

selection: — колір виділення тексту

visual

Змінює колір підсвітки коли користувач виділяє текст мишкою. Виділи текст у демо нижче.

HTML 
<!-- Зелене виділення -->
<p class="selection:bg-emerald-400 selection:text-black">
  Виділи цей текст мишкою
</p>

<!-- Фіолетове виділення -->
<p class="selection:bg-violet-500 selection:text-white">
  А тут фіолетовий
</p>

<!-- На весь блок -->
<div class="selection:bg-amber-300 selection:text-black">
  Весь текст матиме золоте виділення
</div>

05

divide — розділювачі між елементами

layout

Автоматично додає border між дочірніми елементами — не треба вручну ставити border на кожен і прибирати з останнього.

HTML 
<!-- Вертикальний розділювач -->
<div class="divide-y divide-gray-700">
  <div class="p-4">Елемент 1</div>
  <div class="p-4">Елемент 2</div>
  <div class="p-4">Елемент 3</div>
</div>

<!-- Горизонтальний розділювач -->
<div class="flex divide-x divide-gray-700">
  <div class="px-4">Колонка 1</div>
  <div class="px-4">Колонка 2</div>
</div>

<!-- Dashed стиль -->
<ul class="divide-y divide-dashed divide-sky-800">
  <li class="py-3">Пункт списку</li>
</ul>

06

aspect-ratio — збереження пропорцій

layout

Зберігає пропорції елементу незалежно від ширини. Ідеально для відео, зображень, карток. Замінює старий хак з padding-top.

HTML 
<!-- Вбудовані варіанти -->
<div class="aspect-video">16:9 (відео)</div>
<div class="aspect-square">1:1 (квадрат)</div>

<!-- Кастомні -->
<div class="aspect-[4/3]">4:3 (фото)</div>
<div class="aspect-[3/4]">3:4 (портрет)</div>

<!-- Реальний кейс — картка героя -->
<div class="aspect-[3/4] overflow-hidden rounded-xl">
  <img class="w-full h-full object-cover" src="...">
</div>

07

scroll-snap — прокрутка з фіксацією

interact

Браузер автоматично "прилипає" до елементів при скролі. Карусель без бібліотек — тільки CSS класи.

HTML 
<!-- Горизонтальна карусель -->
<div class="flex overflow-x-auto gap-4 snap-x snap-mandatory">
  <div class="snap-start flex-shrink-0 w-64 h-40 rounded-xl">
    Слайд 1
  </div>
  <div class="snap-start flex-shrink-0 w-64 h-40 rounded-xl">
    Слайд 2
  </div>
</div>

<!-- Full-page vertical scroll -->
<div class="h-screen overflow-y-auto snap-y snap-mandatory">
  <section class="snap-start h-screen">Секція 1</section>
  <section class="snap-start h-screen">Секція 2</section>
</div>

08

backdrop-filter — glassmorphism ефект

visual

Розмиває те що знаходиться за елементом. Основа glass-ефекту. Ідеально для navbar, модалок, карток поверх зображень.

HTML 
<!-- Формула glassmorphism -->
<div class="
  backdrop-blur-md
  bg-white/10
  border border-white/20
  rounded-2xl p-6
">
  Скляна картка
</div>

<!-- Navbar з blur -->
<nav class="fixed top-0 w-full backdrop-blur-lg bg-white/70">
  Навігація
</nav>

<!-- Варіанти blur -->
backdrop-blur-sm   4px
backdrop-blur-md   12px
backdrop-blur-xl   24px
backdrop-blur-3xl  64px

09

container + mx-auto — центрування контенту

layout

Стандартний паттерн для обмеження ширини контенту на великих екранах. Фон на всю ширину — контент обмежений.

HTML 
<!-- Кожна секція лендінгу -->
<section class="py-16 md:py-24 bg-gray-900">
  <div class="max-w-7xl mx-auto px-4 md:px-8">
    <!-- контент -->
  </div>
</section>

<!-- max-width варіанти -->
max-w-sm   384px  — форми
max-w-xl   576px  — статті
max-w-3xl  768px  — блог
max-w-5xl  1024px — контент
max-w-7xl  1280px — лендінги
