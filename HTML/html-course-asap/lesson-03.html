01 — Headings

Заголовки H1–H6

В HTML є 6 рівнів заголовків. H1 — найбільший, H6 — найменший. Браузер сам дає їм різні розміри через дефолтні CSS стилі.

Всі шість заголовків
index.html
<!-- Headings -->
<h1>Heading One</h1>
<h2>Heading Two</h2>
<h3>Heading Three</h3>
<h4>Heading Four</h4>
<h5>Heading Five</h5>
<h6>Heading Six</h6>

<!-- Дублювати рядок: Alt+Shift+↓ -->
localhost:5500/index.html

02 — Paragraphs

Параграф <p>

Параграф — для звичайного тексту. Великі блоки тексту, описи, статті — все через <p>. Браузер дає йому дефолтний font-size: 16px.

Параграф і Lorem Ipsum
index.html
<!-- Paragraphs -->
<p>
  Lorem ipsum dolor sit amet,
  consectetur adipiscing elit.
  Sed do eiusmod tempor...
</p>

<!-- Емет трюк: -->
lorem20 → Enter
← генерує 20 слів тексту
   як placeholder!        -->

<!-- Або: lorem50, lorem100 -->

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore.

Дефолт параграфу font-size: 16px + margin зверху і знизу. DevTools покаже це якщо навести на <p>.
03 — Block vs Inline

Найважливіша концепція HTML

ASAP Frontend присвячує цьому окремий розділ. Кожен HTML елемент — або блочний, або рядковий. Це визначає як вони займають місце на сторінці.

Block elements — займають весь рядок
<!-- Block елементи:
     - починаються з нового рядка
     - займають всю доступну ширину  -->

<h1>Заголовок</h1>   ← block
<h2>Підзаголовок</h2>  ← block
<p>Параграф тексту</p>  ← block
<div>Контейнер</div>   ← block
<ul>Список</ul>       ← block

<!-- DevTools: display: block -->
h1 — весь рядок ←
h2 — весь рядок ←
p — весь рядок ←
Кожен block стартує з нового рядка
Inline elements — тільки потрібна ширина
<!-- Inline елементи:
     - НЕ починають новий рядок
     - займають тільки свій вміст -->

<b>жирний</b>       ← inline
<i>курсив</i>        ← inline
<a>посилання</a>    ← inline
<span>текст</span>   ← inline
<img>             ← inline
<button>          ← inline

<!-- Кілька inline поряд
     — вони СТОЯТЬ ПОРЯД!         -->
<b>один</b> <b>два</b> <b>три</b>
один два три
← Три <b> стоять в одному рядку!
Inline поряд, не стартують новий рядок Якщо хочеш inline елементи на різних рядках — обгорни кожен у <div> (block).

<div> — block контейнер для організації
<!-- div ділить контент на секції -->
<!-- Без CSS він невидимий,
     але дуже корисний для layout -->

<div>
  <!-- Все що стосується ім'я -->
  <label>First Name</label>
  <input type="text" />
</div>

<div>
  <!-- Все що стосується email -->
  <label>Email</label>
  <input type="email" />
</div>

<!-- div = block → кожна група
     на своєму рядку ✅             -->
div (block) ставить кожну групу на новий рядок
ASAP Frontend про div: "div stands for division. We use div elements to sort of separate and divide the pieces of content on our page into different little sections." Це основний інструмент організації HTML.
🎯 Завдання — Лекція 03
1. Напиши всі 6 заголовків (h1–h6) з текстом всередині → подивись як зменшується розмір
2. Після h1 додай <p> з lorem20 (Emmet!) → Alt+Shift+F для форматування
3. Напиши три <b>hello</b> підряд — переконайся що вони СТОЯТЬ ПОРЯД (inline!)
4. Обгорни кожен <b> в <div> — тепер вони на різних рядках (block!)
5. Відкрий DevTools → клікни на h1 → знайди в styles "display: block" і "font-size: 32px"
Підказка: щоб написати lorem20 — просто введи текст lorem20 і натисни Tab або Enter
