Links
Anchor Elements

Як робити посилання в HTML. href і https — обов'язково. target="_blank" — нова вкладка. Посилання між HTML файлами через відносні шляхи.

01 — Anchor Element

Тег <a> — посилання

Anchor element — це посилання. Клік → переходиш кудись. Атрибут href каже куди саме. Це inline елемент.

Базовий синтаксис
index.html
<!-- Anchor elements -->

<!-- Emmet: a → Enter → отримуєш: -->
<a href=""></a>

<!-- Заповнюємо: -->
<a href="https://google.com">
  Go to Google
</a>

<!-- Структура:
     <a href="URL">текст посилання</a>
     ↑          ↑              ↑
     тег    куди йдемо    що бачить юзер -->
Go to Google

↑ Синє підкреслене посилання — браузерний дефолт

БЕЗ https — не працює! href="google.com" → шукає файл google.com у твоїй папці. href="https://google.com" → йде в інтернет. Завжди https!
Чому потрібен https — детально
<!-- ❌ НЕ ПРАЦЮЄ — HTML шукає файл google.com у папці -->
<a href="google.com">Google</a>

<!-- ✅ ПРАЦЮЄ — вказуємо повний URL -->
<a href="https://google.com">Google</a>
<a href="https://asapfrontend.com">ASAP</a>
<a href="https://github.com">GitHub</a>

<!-- ASAP Frontend:
     "When you want to send someone to a website,
     you have to make sure to add the https.
     It's going to save you hours of headache." -->
02 — target="_blank"

Відкрити в новій вкладці

За замовчуванням клік на посилання відкриває URL в тій самій вкладці — ти йдеш з своєї сторінки. Якщо хочеш щоб сторінка залишалась відкритою — використовуй target="_blank".

target="_blank" — нова вкладка
<!-- Без target: переходить у цій вкладці -->
<a href="https://google.com">
  Google (та сама вкладка)
</a>

<!-- З target="_blank": нова вкладка! -->
<a
  href="https://google.com"
  target="_blank"
>
  Google (нова вкладка)
</a>

<!-- Або в один рядок: -->
<a href="https://asapfrontend.com"
   target="_blank">ASAP Frontend</a>
Без target:
Клік → йдеш з поточної сторінки → зворотньо через Back кнопку
З target="_blank":
Клік → відкривається нова вкладка → твоя сторінка залишається відкритою
Коли використовувати? Зовнішні посилання (на інші сайти) — зазвичай target="_blank". Посилання між своїми сторінками — без target.
03 — Відносні посилання

Посилання між HTML файлами

Крім зовнішніх сайтів, посилання можуть вести на інші сторінки твого ж сайту. Для цього використовуємо відносні шляхи — без https.

Структура файлів проєкту
html-crash-course/
├── index.html     ← головна сторінка
├── about.html     ← сторінка "Про нас"
└── contact.html   ← сторінка контактів

<!-- index.html: -->
<a href="about.html">Про нас</a>
<a href="contact.html">Контакти</a>

<!-- about.html — повертаємось назад: -->
<a href="index.html">← На головну</a>

<!-- Ключове: НЕ додаємо https! -->
<!-- HTML шукає файл у тій же папці -->
Чому без https? href="about.html" — HTML шукає файл about.html в тій самій папці що й поточний HTML файл. Це "відносний" шлях.
Без target="_blank" Для переходів між своїми сторінками — не додаємо target="_blank". Користувач залишається "всередині" твого сайту.
Як створити about.html — покрокова практика
Крок 1: Створи about.html
<!-- В VS Code: Explorer → New File → about.html -->
<!-- Потім введи ! → Enter (Emmet boilerplate) -->

<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <title>About Us</title>
</head>
<body>

  <h1>About Us Page</h1>

  <!-- Кнопка повернення назад: -->
  <a href="index.html">← Back to Index</a>

</body>
</html>
Крок 2: Додай посилання в index.html
<!-- Anchor elements -->

<!-- Зовнішнє посилання (з https, нова вкладка): -->
<a href="https://asapfrontend.com"
   target="_blank">
  ASAP Frontend
</a>

<!-- Внутрішнє посилання (відносне, без https): -->
<a href="about.html">About Us</a>
<a href="contact.html">Contact</a>
04 — Підсумок

Всі варіанти посилань

Шпаргалка anchor elements
ТипhreftargetРезультат
Зовнішній (та вкладка) https://site.com — Переходить у цій вкладці
Зовнішній (нова вкладка) https://site.com _blank Відкриває нову вкладку
Внутрішній (своя сторінка) about.html — Перехід на свою сторінку

🎯 Завдання — Лекція 04
1. В index.html створи посилання на asapfrontend.com з target="_blank"
2. Створи contact.html з boilerplate (!) → додай h1 "Contact Page"
3. В index.html додай посилання href="contact.html" → перевір що воно працює
4. В contact.html додай href="index.html" для повернення назад
5. Спробуй href="google.com" БЕЗ https → переконайся що не працює → виправ
6. Два посилання поряд — вони стоять в одному рядку (inline!) — перевір
Підказка: якщо посилання не відкриває сторінку — перевір чи правильна назва файлу і чи він в тій самій папці
