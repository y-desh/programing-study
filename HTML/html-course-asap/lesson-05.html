Lists
Списки

Два типи списків: ul (unordered) і ol (ordered). List items через <li>. Коли що використовувати. Вкладені списки.

01 — Unordered List

<ul> — список без порядку

ASAP Frontend: "We use unordered list when the order of our list doesn't really matter." Маркований список — 99% випадків у реальній розробці.

Синтаксис ul
index.html
<!-- Lists -->

<!-- Unordered list: -->
<h1>Grocery List</h1>

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Grapes</li>
</ul>

<!-- ul = Unordered List (батько)
     li = List Item (дочірній)
     
     ul → block
     li → block  -->

<!-- Порядок НЕ важливий:
     Виноград → Банани → Яблука
     — однаково валідно ✅         -->

Grocery List

  • Apples
  • Bananas
  • Grapes
• Bullet points за замовчуванням Браузер автоматично додає • перед кожним li в ul. Через CSS можна замінити або прибрати (list-style: none).
02 — Ordered List

<ol> — список з порядком

ASAP Frontend: "If the order of your list really does matter, that is when you would be using an ordered list." Браузер автоматично нумерує.

Синтаксис ol — рейтинг, кроки, інструкції
index.html
<!-- Ordered list: -->
<h1>Leaderboard</h1>

<ol>
  <li>Usain Bolt</li>   ← 1 місце
  <li>Ronaldo</li>     ← 2 місце
  <li>Messi</li>       ← 3 місце
</ol>

<!-- Структура ОДНАКОВА з ul:
     ol замість ul — і все!
     
     Порядок ВАЖЛИВИЙ:
     Bolt завжди перший (1) ✅ -->

Leaderboard

  1. Usain Bolt
  2. Ronaldo
  3. Messi
1. 2. 3. — автоматично Браузер сам нумерує li всередині ol. Якщо перемістити Messi на перше місце — він автоматично стане 1.
03 — ul vs ol

Коли що використовувати

Порівняльна таблиця
ТегМаркуванняКолиПриклад
<ul> Bullet points • Порядок НЕ важливий Список продуктів, навігація сайту, функції продукту
<ol> Числа 1. 2. 3. Порядок ВАЖЛИВИЙ Рейтинг, інструкція крок за кроком, рецепт
ASAP Frontend: "Typically we use unordered list when the order of our list doesn't really matter. This is going to be the case 99% of the time." В реальних проєктах ul зустрічається набагато частіше — навбари, картки, меню — всі через ul.
Реальні приклади зі сайтів
ul — навігаційне меню
<nav>
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/about">About</a></li>
    <li><a href="/blog">Blog</a></li>
  </ul>
</nav>
<!-- CSS зробить це горизонтальним
     і без bullet points -->
ol — кроки інструкції
<h2>Як встановити VS Code</h2>
<ol>
  <li>Завантаж з code.visualstudio.com</li>
  <li>Відкрий завантажений файл</li>
  <li>Дотримуйся інструкцій</li>
  <li>Запусти VS Code</li>
</ol>
<!-- Порядок КРИТИЧНО важливий! -->
