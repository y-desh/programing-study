Forms
Форми

form, label, input (5 типів), textarea, select + option, button. placeholder атрибут. Самозакриваючі теги на практиці. Div для групування полів.

01 — Form Element

<form> — контейнер для всіх полів

Form — обгортка для всіх елементів введення. ASAP Frontend: "We put all of the information that we want the user to fill in inside of a form."

Базова структура форми
<!-- Forms -->

<form>
  <!-- Тут всі поля форми -->
</form>

<!-- Емет: form → Enter → отримуєш:
     <form action=""></form>
     
     action — що робити після submit.
     Потрібен JavaScript для обробки.
     Поки що можна видалити action.    -->

<!-- Все що стосується однієї
     групи — обгортаємо в div: -->
<form>
  <div>
    <!-- Ім'я: label + input -->
  </div>
  <div>
    <!-- Email: label + input -->
  </div>
</form>
Навіщо <form>? Семантика — браузер знає що це форма. Enter автоматично Submit. Атрибут action каже куди відправити дані. Без JavaScript — форма нічого не зробить.
div всередині form ASAP Frontend: "This helps to keep everything separated and easy to understand." Кожна логічна група (label + input) в окремому div.
02 — Label + Input

Мітка і поле введення

Label — текстовий підпис. Input — само поле. Вони завжди разом. Input — самозакриваючий тег з атрибутом type.

label і input — синтаксис
<form>
  <div>
    <!-- label — підпис поля -->
    <label>First Name</label>

    <!-- input — поле вводу.
         Самозакриваючий тег!
         / перед > — best practice -->
    <input type="text" />
  </div>

  <div>
    <label>Email</label>
    <input type="email" />
  </div>
</form>

<!-- label і input — inline!
     Через div вони на різних рядках. -->
Всі типи input — type атрибут
Різні type для різних даних
<!-- text — звичайний текст -->
<input type="text" />

<!-- email — з валідацією формату email -->
<input type="email" />
<!-- Якщо написати не-email → браузер видасть помилку -->

<!-- password — текст прихований ● ● ● -->
<input type="password" />

<!-- number — тільки числа, +/- кнопки -->
<input type="number" />

<!-- date — calendar picker -->
<input type="date" />

<!-- checkbox — чекбокс (on/off) -->
<input type="checkbox" />

<!-- radio — вибір одного з кількох -->
<input type="radio" name="gender" />
text:
email:
password:
number:
date:
Навіщо type="email" якщо це просто текст? ASAP Frontend: "The email input gives us some extra validation to make sure what we are typing is actually an email. If you fill in an invalid email — we're going to get a red squiggly line and an error." Безкоштовна валідація від браузера!
03 — placeholder

Текст-підказка всередині поля

placeholder — зникає при введенні
<!-- placeholder — текст в порожньому полі -->
<input
  type="text"
  placeholder="Type in your first name"
/>

<input
  type="email"
  placeholder="Enter your email"
/>

<input
  type="password"
  placeholder="Min 8 characters"
/>

<!-- Коли починаєш вводити —
     placeholder зникає.
     Коли очищуєш — повертається. -->
04 — textarea, select, button

Спеціальні елементи форми

<textarea> — багаторядкове поле
<!-- textarea — НЕ самозакриваючий! -->
<!-- Для довгих повідомлень, email, відгуків -->

<div>
  <label>Message</label>
  <textarea></textarea>
</div>

<!-- Відмінності від input:
     1. Має end tag (НЕ самозакривається)
     2. Немає type атрибуту
     3. Можна ресайзити мишею (за кутик)
     4. Для довгого тексту               -->

<select> + <option> — випадаючий список
<!-- select — dropdown (вибір зі списку) -->
<!-- option — кожен варіант вибору -->
<!-- value — значення для JavaScript -->

<div>
  <label>Clothing Size</label>

  <select>
    <option value="xs">XS</option>
    <option value="s">S</option>
    <option value="m">M</option>
    <option value="l">L</option>
  </select>
</div>

<!-- value важливий коли обробляємо
     форму через JavaScript!           -->

<button> — кнопка submit
<!-- button — для відправки форми -->
<button>Submit</button>

<!-- Без JavaScript нічого не відбувається.
     Але браузер все одно показує кнопку
     і вона готова приймати обробку.   -->

<!-- type="submit" — явна вказівка: -->
<button type="submit">Submit Form</button>
Повна форма — все разом
index.html — повна форма

<form>

  <div>
    <label>First Name</label>
    <input type="text" placeholder="Enter first name" />
  </div>

  <div>
    <label>Email</label>
    <input type="email" placeholder="Enter email" />
  </div>

  <div>
    <label>Password</label>
    <input type="password" placeholder="Min 8 chars" />
  </div>

  <div>
    <label>Message</label>
    <textarea></textarea>
  </div>

  <div>
    <label>Size</label>
    <select>
      <option value="s">S</option>
      <option value="m">M</option>
      <option value="l">L</option>
    </select>
  </div>

  <button type="submit">Submit</button>

</form>
