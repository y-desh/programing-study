01 — Button

<button> — кнопка

Кнопки — всюди на сайтах. "Submit", "Buy Now", "Login" — все через <button>. Це inline елемент, але часто обгортається в div для layout.

Синтаксис і поведінка
index.html
<!-- Buttons -->

<!-- Базова кнопка -->
<button>Click Me</button>

<!-- button — INLINE елемент!
     Стоїть поряд з іншими inline.
     
     Щоб бути на новому рядку —
     обгорни в div (block):      -->
<div>
  <button>Click Me</button>
</div>

<!-- Різні типи кнопок: -->
<button type="submit">Submit Form</button>
<button type="button">Just a Button</button>
<button type="reset">Reset Form</button>

02 — Span

<span> — inline контейнер для стилізації

Span — як div, але inline. Використовується щоб виділити частину тексту і застосувати до неї CSS або отримати через JavaScript.

Навіщо span — реальний приклад
index.html
<!-- Spans -->

<!-- Хочемо частину тексту зробити
     червоною через CSS:          -->
<p>
  Land a job with
  <span class="highlight">
    ASAP Front End
  </span>
</p>

<!-- В CSS (потім): -->
.highlight {
  color: red;
}
<!-- → "ASAP Front End" стане червоним
     а решта тексту залишиться звичайною -->

<!-- Без span — не можна виділити
     лише частину параграфу!            -->

Land a job with ASAP Front End


↑ span з class="highlight" + CSS color: red

Without a span you cannot target only part of text

class атрибут — зв'язок HTML і CSS
<!-- class — для цільового CSS таргетингу -->
<span class="highlight">текст</span>

<!-- В CSS файлі: -->
/* .highlight { color: red; } */

<!-- class можна давати БУДЬ-ЯКОМУ елементу: -->
<h1 class="main-title">Title</h1>
<p class="description">Text</p>
<div class="card">...</div>
<button class="btn-primary">Click</button>

<!-- Кілька класів на один елемент: -->
<button class="btn btn-primary btn-large">
  Big Primary Button
</button>

<!-- ASAP Frontend: "We would give this
     a class name. Using CSS, we would go
     in and target this individual span." -->
div vs span — коли що
ЕлементТипКоли використовувати
<div> block Групуємо блочні секції. Форма, картка, секція сторінки.
<span> inline Виділяємо частину тексту або inline елементів для CSS.

Повна шпаргалка — всі теги курсу
HTML Cheat Sheet
<!-- ═══ СТРУКТУРА ══════════════════════════ -->
<!DOCTYPE html>   ← завжди перший рядок
<html>             ← корінь документа
<head>             ← метадані (невидиме)
<body>             ← видимий вміст
<title>            ← заголовок вкладки
<meta>             ← кодування, viewport

<!-- ═══ ЗАГОЛОВКИ ═════════════════════════ -->
<h1></h1>          ← найбільший (32px)
<h2></h2>          ← 24px
<h3></h3>          ← 18px
<h4> <h5> <h6>    ← менші

<!-- ═══ ТЕКСТ ══════════════════════════════ -->
<p></p>             ← параграф (block)
<b></b>             ← жирний (inline)
<i></i>             ← курсив (inline)
<span></span>        ← inline контейнер

<!-- ═══ ПОСИЛАННЯ ═════════════════════════ -->
<a href="https://..." target="_blank">
<a href="page.html">   ← внутрішнє

<!-- ═══ СПИСКИ ════════════════════════════ -->
<ul>               ← без порядку (• • •)
<ol>               ← з порядком (1. 2. 3.)
<li></li>          ← пункт списку (block)

<!-- ═══ ФОРМИ ════════════════════════════  -->
<form></form>        ← контейнер форми
<label></label>      ← підпис поля
<input type="text"     />  ← текст
<input type="email"    />  ← email
<input type="password" />  ← пароль
<input type="number"   />  ← число
<input type="date"     />  ← дата
<textarea></textarea>  ← великий текст
<select></select>      ← dropdown
<option value="x"></option> ← варіант
<button></button>      ← кнопка

<!-- ═══ ЗОБРАЖЕННЯ ════════════════════════ -->
<img src="url" alt="опис" />

<!-- ═══ LAYOUT ════════════════════════════ -->
<div></div>          ← block контейнер
<br />             ← перенос рядка

<!-- ═══ АТРИБУТИ ══════════════════════════ -->
class="назва"      ← для CSS таргетингу
id="унікально"     ← для JavaScript
href="..."         ← куди посилання
src="..."          ← звідки зображення
alt="..."          ← опис зображення
target="_blank"   ← нова вкладка
placeholder="..." ← підказка в полі
type="..."         ← тип input'у
value="..."        ← значення option
Block vs Inline — фінальна таблиця
BLOCK — новий рядок, повна ширина
<h1> – <h6>
<p>
<div>
<ul>, <ol>, <li>
<form>
INLINE — поряд, тільки свій розмір
<a>
<span>
<b>, <i>
<img>
<button>, <input>
