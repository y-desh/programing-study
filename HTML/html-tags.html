🏷️ HTML Теги

Повна шпаргалка — від структури до форм

📐 Структура сторінки
<html>Корінь документа
<head>Мета-інформація (не видима)
<body>Весь видимий контент
<meta>charset, viewport, SEO
<title>Назва у вкладці браузера
<link>Підключення CSS, favicon
<script>Підключення JS
<style>CSS прямо в HTML
🏗️ Семантичні блоки
<header>Шапка сайту або секції
<main>Головний контент — один на сторінці
<footer>Підвал сайту або секції
<nav>Навігаційне меню
<section>Тематична секція з заголовком
<article>Самостійний незалежний контент
<aside>Бічна панель, доповнення
<div>Нейтральний блок без змісту
📝 Текст та типографія
<h1>–<h6>Заголовки, h1 — один на сторінці
<p>Параграф тексту
<span>Інлайн контейнер без змісту
<strong>Важливий текст (жирний)
<em>Акцентований текст (курсив)
<br>Перенос рядка
<hr>Горизонтальна лінія
<blockquote>Цитата з іншого джерела
<code>Інлайн код
<pre>Блок коду зі збереженням пробілів
<mark>Виділений / підсвічений текст
<time>Дата або час — datetime для SEO
<address>Контактна інформація
<abbr>Абревіатура з розшифровкою
🔗 Посилання та медіа
<a>Посилання — href, target
<img>Зображення — alt обов'язковий
<picture>Адаптивні зображення різних форматів
<video>Відео — autoplay, controls, loop
<audio>Аудіо — controls
<source>Джерело для video / audio / picture
<iframe>Вбудована зовнішня сторінка
<svg>Векторна графіка прямо в HTML
<canvas>Полотно для малювання через JS
<figure>Зображення або медіа з підписом
<figcaption>Підпис до figure
📋 Списки
<ul>Невпорядкований список (крапки)
<ol>Впорядкований список (цифри)
<li>Елемент списку
<dl>Список визначень
<dt>Термін у dl
<dd>Визначення терміну у dl
📊 Таблиці
<table>Таблиця
<thead>Шапка таблиці
<tbody>Тіло таблиці
<tfoot>Підвал таблиці
<tr>Рядок таблиці
<th>Заголовок стовпця
<td>Комірка з даними
📬 Форми
<form>Форма — action + method
<input>Поле вводу — type визначає тип
<label>Підпис до інпуту — for="id"
<button>Кнопка — submit / button / reset
<textarea>Багаторядкове поле тексту
<select>Випадаючий список
<option>Варіант у select
<optgroup>Група варіантів у select
<fieldset>Група полів форми
<legend>Заголовок для fieldset
<datalist>Підказки-автодоповнення для input
<progress>Прогрес-бар
<meter>Шкала значення (HP, рейтинг)
🔢 Типи input (type=)
textЗвичайний текст
emailEmail з валідацією
passwordПрихований текст
numberТільки числа
telТелефон
urlПосилання
searchПошук
checkboxЧекбокс
radioОдин з варіантів
rangeСлайдер
dateВибір дати
timeВибір часу
colorПалітра кольорів
fileЗавантаження файлу
hiddenПрихований (для JS)
submitКнопка відправки
🎛️ Інтерактивні елементи
<details>Розкривний блок (акордеон без JS)
<summary>Заголовок для details
<dialog>Нативне модальне вікно
